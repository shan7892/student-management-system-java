# Student Management System

A Java-based Student Management System with a graphical user interface (GUI) and MySQL database integration.

This project manages student and teacher records while demonstrating Object-Oriented Programming concepts, database connectivity, and CRUD operations.

## 🚀 Features

- Add student records
- Update student information
- Delete student records
- View student details
- Manage teacher records
- GUI-based user interface
- Database connectivity using MySQL

## 🛠 Technologies Used

- Java
- Java Swing (GUI)
- MySQL
- JDBC
- Object-Oriented Programming (OOP)
